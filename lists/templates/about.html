Academic Projects

  • Data analysis and management (TM351, Tutors: Dr. Tony Burrows and Ms. J Buxton, 2016):

    An analysis of bat hibernation and roost data using K-means clustering and silhouette analysis. Statistical differences between the two datasets were also explored including the use of the Chi-squared test. Consideration was also given to data protection.
    Technologies used: Python, numpy, pandas, matplotlib, scipy, seaborn and sklearn machine learning package.

  • Distributed and concurrent systems (M363, Tutor: Mr. D Oddie, 2015):

    Throughout the course several elements of a music store auction site were developed included database logic, chat room and front end pages.

    Technologies used: Java EE, EJBs, servlets. Glassfish3

  • Software engineering (TM354, Tutor: Dr. C Thomson 2016):

    Group projects included designing a management system for a boat club. With activity diagram, conceptual model, performance and usability scenarios considered.

    This course involved: Java and an agile software design approach based on scrum and UML.

  • Renewable energy systems (T313, Tutor: Mr J R Bradley 2015):

    An independent desk study was undertaken to advice on implementation of fabrics and renewable energy system improvements to a residential home. NHER plan assessor was used and a detailed financial assessment taking into account NPV and IRR was undertaken. Advice was then presented based on this analysis taking into account financial outcomes and CO2 emissions.
